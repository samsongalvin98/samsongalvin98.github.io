3D Printing File Upload Requests

Submit a request for an FDM 3D print. Include your part file and enough detail to estimate time, material, and setup.

Note: This site is hosted on GitHub Pages, so uploads must go to your separate backend server. Configure your backend endpoint in assets/js/lab-form-endpoints.js to enable submissions.
Accepted: STL, STEP, ZIP. You can upload multiple files.
Used only for STL estimates (STL files don’t store units).
Estimates are approximate and intended for quick quoting only.
